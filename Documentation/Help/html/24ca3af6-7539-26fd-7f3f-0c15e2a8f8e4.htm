TAD8811_Cmd メソッド

TAD8811_Cmd 型は下記のメンバーを公開します。

メソッド
  名前説明
AlarmRest
アラーム解除します。
CloseDevice
TAD8811が接続済みのCOMポートをCloseします。
Equals
指定した Object が、現在の Object と等しいかどうかを判断します。
(Objectから継承)
GetFeedback
フィードバックデータ（位置・速度・電流）を取得します。
GetHashCode
特定の型のハッシュ関数として機能します。
(Objectから継承)
GetType
現在のインスタンスの Type を取得します。
(Objectから継承)
HardStop
即停止します。
InitTAD8811

TAD8811のパラメータを初期化します。

ID69（コントロールスイッチ）：サーボオフ時の偏差リセット有効＋プロファイル動作完了時のフラグ自動クリア有効

ID209（アラームマスク）：多回転アラーム検出禁止

OpenDevice
TAD8811がUSB接続済みならCOMポートをOpenします。
PositionReset
TAD8811にポジションリセット指令を発行します。
ReadParam
パラメータを読み込みます。1個のパラメータを読み込みます。
ReadParamAll
全てのパラメータを読み込みます。一度に256個のパラメータを読み込みます。
RequestCommandCode
リクエストコマンド（特殊指令）を発行します。
ServoOff
サーボオフ指令を発行します。
ServoOn
サーボオン指令を発行します。
SetDirection
正回転方向を設定します。
SmoothStop
減速停止します。
StartJog
一定速送り（速度制御）を開始します。
StartProfile
位置決め動作を開始します。
ToString
現在の Object を表す String を返します。
(Objectから継承)
WaitHoming
原点復帰完了（原点到達）まで待機します。
WaitInpositon
位置決め完了（インポジション到達）まで待機します。
WriteParam
パラメータを変更します。
